FreeCodeCamp Project: URL Shortener Microservice

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

https://short-my-url.herokuapp.com/www.google.com
https://short-my-url.herokuapp.com/www.foo.com

Example creation output

{ "original_url":"www.foo.com",
"short_url":"https://short-my-url.herokuapp.com/9241" }

Usage:

https://short-my-url.herokuapp.com/9241

Will redirect to:

www.foo.com
